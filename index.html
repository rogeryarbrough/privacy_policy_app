Privacy Policy

Effective Date: January 7, 2026

What We Collect

This app accesses Facebook Page data including follower counts and basic Page information through the Facebook Graph API. We only access Pages where the account holder is an administrator.

How We Use Data

Data is used solely for displaying Page follower statistics to the Page administrator. No data is stored, shared, or used for any other purpose.

No Data Storage

We do not store, log, or retain any Facebook user data or Page data beyond the immediate API response needed to display results.

Permissions Used

  • pages_show_list - To list Pages the user administers
  • pages_read_engagement - To read Page follower counts and insights

Contact

For questions about this policy, contact the app administrator through Facebook Developer tools.

This app complies with Meta Platform Policy and applicable privacy laws.
